# ChitChat - Chat with friends!
A chatapp made with Express, Node.js, and Socket.io

by Syrrel Cabiling

#### Set up
1. Clone the repo to anywhere (i. Documents)
   ```
   git clone https://github.com/syrrelCabiling/ChatApp.git
   ```
   cd ChatApp
   ```
2. Load through localhost:3030
   ```
